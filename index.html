Each IIFE should expose, in its public interface, a method named `add{ingredient}` (e.g. `addMeat` or `addVeggies`) that accepts a single argument. That argument's value should be the ingredient selected by the user. You should also create an additional JavaScript file that handles interacting with the form elements and determining which method should be called.
